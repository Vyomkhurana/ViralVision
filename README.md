# ViralVision 

ViralVision is a machine learning project that predicts whether a YouTube video will be **Low**, **Medium**, or **Viral** based on:

- Title
- Description
- Tags
- Channel and video metadata

## Tech Stack

- Python
- YouTube Data API v3
- Pandas, NumPy
- Scikit-learn
- NLTK
- Streamlit

## Project Structure

- `src/` – data collection, preprocessing, model training, prediction scripts
- `data/` – raw and processed datasets (ignored from Git)
- `notebooks/` – Jupyter notebooks for experiments
- `app/` – Streamlit web app
